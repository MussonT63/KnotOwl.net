Misc
  • How can I see all my Awards, Trainings, etc.
  • my.Scouting, My Profile - on the right of the "Personal Information" line, select "Print".

  • How can I generate a PersonProfileReport with Awards, Trainings, etc for others?
  • This can only be done by Key3. Go to my.Scouting, Unit/Roster. Select the one person you want the report for. Select Print/"Unit Advancement Details Report" (yep, even for adults!).
